# Installation

Step 1 (Clone or Download)
```
git clone https://github.com/jaleelx98/Bad-Robo.git
```
Step 2 (Open Bad-Robo Drectory)
```
cd Bad-Robo
```

Step 4 (Install BadRobo Requirements > This Take Some time 10-15 mints )
